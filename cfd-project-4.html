2D Viscous Burgers Equation

Introduction

The 2D viscous Burgers equation is a fundamental model problem in computational fluid dynamics that combines nonlinear advection with viscous diffusion. This equation serves as a bridge between linear diffusion problems and fully nonlinear Navier-Stokes equations, making it an excellent benchmark for testing numerical schemes. The equation is given by:

\[ \frac{\partial u}{\partial t} + u\frac{\partial u}{\partial x} + u\frac{\partial u}{\partial y} = c_1\frac{\partial^2 u}{\partial x^2} + c_2\frac{\partial^2 u}{\partial y^2} \]

where \(u(x,y,t)\) is the solution field and \(c_1, c_2\) are viscosity coefficients. In this assignment, we implement three different finite difference schemes to compare their accuracy, stability, and convergence behavior.

Problem Setup

Domain and Boundary Conditions

Domain: [0,1] × [0,1], Time: [0, 10]

Coefficients: \(c_1 = 1.0\), \(c_2 = 0.5\), \(\gamma = 0.02\)
Initial Condition: \(u(x,y,0) = 0\)
Boundary Conditions:
\(u(x,0,t) = 0\), \(u(x,1,t) = 1\)
\(u(0,y,t) = 0\), \(u(1,y,t) = 1\)

Governing PDE:

\[ \frac{\partial u}{\partial t} + u\frac{\partial u}{\partial x} + u\frac{\partial u}{\partial y} = c_1\frac{\partial^2 u}{\partial x^2} + c_2\frac{\partial^2 u}{\partial y^2} \]

Grid Configurations:

  1. \(N_x = N_y = 30\)
  2. \(N_x = N_y = 12\)
  3. \(N_x = 30, N_y = 15\)

Methodology

Nature of PDE: 2nd order, nonlinear, parabolic.

Numerical Schemes

Three different finite difference schemes were implemented and compared:

1. FTCS Scheme (Forward-Time Central-Space)

Explicit scheme with forward difference in time and central differences in space.

\[ \frac{u_{i,j}^{n+1}-u_{i,j}^n}{\Delta t} + u_{i,j}^n\frac{u_{i+1,j}^n-u_{i-1,j}^n}{2\Delta x} + u_{i,j}^n\frac{u_{i,j+1}^n-u_{i,j-1}^n}{2\Delta y} = c_1\frac{u_{i+1,j}^n-2u_{i,j}^n+u_{i-1,j}^n}{\Delta x^2} + c_2\frac{u_{i,j+1}^n-2u_{i,j}^n+u_{i,j-1}^n}{\Delta y^2} \]

2. FTUS Scheme (Forward-Time Upwind Space)

Explicit scheme with upwind differencing for convection terms to improve stability.

\[ \frac{u_{i,j}^{n+1}-u_{i,j}^n}{\Delta t} + u_{i,j}^n\frac{u_{i,j}^n-u_{i-1,j}^n}{\Delta x} + u_{i,j}^n\frac{u_{i,j}^n-u_{i,j-1}^n}{\Delta y} = c_1\frac{u_{i+1,j}^n-2u_{i,j}^n+u_{i-1,j}^n}{\Delta x^2} + c_2\frac{u_{i,j+1}^n-2u_{i,j}^n+u_{i,j-1}^n}{\Delta y^2} \]

3. Mixed Scheme (Implicit-Explicit)

Semi-implicit scheme with explicit treatment of nonlinear terms and implicit treatment of diffusion, solved using TDMA (Thomas Algorithm).

Order of Accuracy

FTCS and Mixed Schemes: \(O(\Delta t, \Delta x^2, \Delta y^2)\)

FTUS Scheme: \(O(\Delta t, \Delta x, \Delta y)\)

Stability Analysis

Von-Neumann stability analysis was performed for all three schemes. The FTCS and FTUS schemes are conditionally stable, while the mixed scheme shows improved stability properties with proper selection of time step and convergence tolerance. Stability requires:

  • FTCS and FTUS: \(r_1 + r_2 \leq 0.5\)
  • Mixed Scheme: Requires \(\Delta t \ll \text{tolerance}\) for stable convergence
